# HinchaBolasBot

### Instalacion:

#### npm install

### Run server

#### npm start


## Para realizar pruebas: 
  - crear un nuevo BOT desde el @BotFather 
  - solicitar el token 
  - crear archivo config/dev.js exportando el token creado 
  - realizar las pruebas desde el BOT creado para prueba.
