# HinchaBolasBot

### Instalacion:

#### npm install

### Run server

#### npm start


## Para realizar pruebas: 
  - crear un nuevo BOT desde el @BotFather 
  - solicitar el token 
  - reemplazarlo en bot.js  
  - realizar las pruebas desde el BOT creado para prueba.
